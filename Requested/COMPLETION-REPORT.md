## ðŸŽ¯ Key Design Decisions

### Why OLED Dark Theme?
- **Power Efficiency**: Black pixels consume zero power on OLED displays
- **Eye Comfort**: Reduced eye strain in low-light environments
- **Modern Aesthetic**: Clean, technical look for professional app
- **Performance**: No complex shadows or gradients

### Why No Rounded Corners?
- **Modern/Technical**: Sharp geometric aesthetic
- **Consistency**: Easier to maintain across all components
- **Performance**: No border-radius calculations
- **Accessibility**: Simpler focus ring styling

### Why System Fonts Only?
- **Performance**: Zero font file downloads
- **Native Feel**: Matches user's operating system
- **Accessibility**: OS fonts optimized for readability
- **Speed**: Instant text rendering

### Why Tailwind CSS?
- **Productivity**: Build 5x faster with utility classes
- **Type-Safe**: Perfect integration with TypeScript
- **Customizable**: Complete control over design system
- **Optimized**: Automatically purges unused CSS
